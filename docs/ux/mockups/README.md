# Tervezési artifactok (wireframe-ek)

A fejlesztés során készültek egyszerű wireframe-ek a főbb funkciókhoz (pl. Home, Course nézet, jegyzet megjelenítés), azonban ezek nem kerültek külön design tool-ba (pl. Figma), hanem gyors vázlatként szolgáltak a funkcionalitás megtervezéséhez.

A végleges UI több ponton eltér ezektől a wireframe-ektől, mivel a fejlesztés során a hangsúly az egyszerűbb használhatóságra és jobb felhasználói élményre került (pl. navigáció átalakítása, elemek átrendezése).

A wireframe-ek főként a struktúra és funkciók meghatározására szolgáltak, míg a vizuális design iteratív módon, implementáció közben alakult ki.
